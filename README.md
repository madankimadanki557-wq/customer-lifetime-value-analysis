# customer-lifetime-value-analysis# Customer Lifetime Value Analysis

## Objective
To identify high-value customers and understand customer purchasing behavior.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

## Analysis Performed
- Customer Lifetime Value Calculation
- Top Customer Identification
- Customer Segmentation
- Data Visualization

## Key Findings
- Calculated CLV for 500 customers.
- Identified top 10 high-value customers.
- Analyzed distribution of customer value.
- Visualized customer behavior using charts.

## Conclusion
Customers with higher purchase frequency, larger order values, and longer customer relationships generate the highest lifetime value.
